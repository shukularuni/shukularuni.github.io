Access

SSH: ssh oracle11@oracle.underthewire.tech -p 22

Password: yondu

Info

The password for oracle12 is the drive letter associated with the mapped drive that this user has.

NOTE:
– Submission should be one letter and lowercase.

▼ HINT:

The registry is awesome!

Theory

To get the password, as the description says, we have to get the letter of the drive that is associated to the drive that we have (the C: drive). To do this, we can use the get-smbmapping command, which just enumerates all the mapped network drives. That's it, pretty simple huh? So let's get to the shell. The command we'll use is just this:

Get-SmbMapping

Solution

Now that we are inside of the shell, we'll run the command to see all mapped drives:

PS C:\users\Oracle11\desktop> Get-SmbMapping

Status      Local Path Remote Path            
------      ---------- -----------            
Unavailable M:         \\127.0.0.1\WsusContent

Yup, that's it. Now just make the drive letter lowercase and remove its colon:

m
