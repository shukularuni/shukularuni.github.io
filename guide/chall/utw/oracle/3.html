Access

SSH: ssh oracle3@oracle.underthewire.tech -p 22

Password: 2f5c4

Info

The password for oracle4 is the date that the system logs were last wiped as depicted in the event logs on the desktop.

NOTE:
– The format for the password is 2 digit month, 2 digit day, 4 digit year. Ex: 5 Jan 2015 would be 01/05/2015.

▼ HINT:

https://msdn.microsoft.com/en-us/powershell/reference/5.1/microsoft.powershell.diagnostics/get-winevent

Theory

To get the password, as the description says, we have to get the date from when the system was last wiped, well "clear" would be the type of word we'd have to look out for in the system logs. Although, I have no idea what type of log it might be or what the file is even called, so let's enter to the shell to know what to continue with.

Solution

Now that we are inside of the shell, let's view the directory to see what the logs are:

PS C:\users\Oracle3\desktop> dir

    Directory: C:\users\Oracle3\desktop


Mode                LastWriteTime         Length Name
----                -------------         ------ ----
-a----        8/30/2018   5:52 AM        1118208 Oracle3_Security.evtx

Oh it's just a normal Windows Event file, well that makes things easier. We can use this command to view all logs that are logged in this file:

PS C:\users\Oracle3\desktop> Get-Winevent -Path .\Oracle3_Security.evtx


   ProviderName: Microsoft-Windows-Security-Auditing

TimeCreated                     Id LevelDisplayName Message
-----------                     -- ---------------- -------
5/18/2017 11:38:23 PM         4624 Information      An account was successfully logged on....
5/18/2017 11:38:23 PM         4672 Information      Special privileges assigned to new logon....
5/18/2017 11:38:23 PM         4634 Information      An account was logged off....
5/18/2017 11:37:38 PM         4624 Information      An account was successfully logged on....

...

To search for any clearing of system logs we can wildcard search the word "clear" to make it easier to find this log:

PS C:\users\Oracle3\desktop> Get-Winevent -Path .\Oracle3_Security.evtx | Where-Object {$_.Message -Like "*clear*"}


   ProviderName: Microsoft-Windows-Eventlog

TimeCreated                     Id LevelDisplayName Message
-----------                     -- ---------------- -------
5/9/2017 11:36:05 PM          1102 Information      The audit log was cleared....

Would you look at that, there's only one log with the word. So let's grab that TimeCreated date and add a few zeros on the day and month to have it be the format requested from the description, and we get this:

05/09/2017
