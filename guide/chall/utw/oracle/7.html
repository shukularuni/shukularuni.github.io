Access

SSH: ssh oracle7@oracle.underthewire.tech -p 22

Password: t-50_97

Info

The password for oracle8 is the name of the domain that a trust is built with PLUS the name of the file on the user’s desktop.

NOTE:
– The password will be lowercase no matter how it appears on the screen.
– If the name of the trust is “blob” and the file on the desktop is named “1234”, the password would be “blob1234”.

Theory

To get the password, as the description says, we have to get name of whatever has a trust built with the system, we can do this by using Active Directory, specifically the Get-ADTrust command to get all the trusted things of the system, just like this with the -Filter option for any, to view all trusted thingies of the system:

Get-ADTrust -Filter *

Solution

Now that we are inside of the shell, let's see the name of the file:

PS C:\users\Oracle7\desktop> dir

    Directory: C:\users\Oracle7\desktop


Mode                LastWriteTime         Length Name
----                -------------         ------ ----
-a----        8/30/2018  10:50 AM              0 111

Now let's see all trusted things from Acive Directory:

PS C:\users\Oracle7\desktop> Get-ADTrust -Filter *


Direction               : Outbound
DisallowTransivity      : True

Oh there we go, we have a trusted thingie with the Name of "multiverse", now we'll just join it with the name of the file in the Desktop to get the password:

multiverse111
