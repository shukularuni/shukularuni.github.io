Access

SSH: ssh oracle12@oracle.underthewire.tech -p 22

Password: m

Info

The password for oracle13 is the IP of the system that this user has previously established a remote desktop with.

▼ HINT:

The registry is awesome!

Theory

To get the password, as the description says, we have to get an IP that has been established on the system to a remote desktop. This type of connection stuff (SSH,SSL,etc) would normally be stored on the terminal or in PowerShell, the thing is; I don't know how the terminal or PowerShell are called on the system registry, so we'll have to look for them on the Microsoft Apps folder on the registry, which is found inside the Software folder of the User's registry, we've used registry previously a lot so this might be easy. So the command to view the Microsoft registry would be this:

Get-ChildItem "HKCU:\Software\Microsoft"

Solution

Now that we are inside of the shell, we'll run the command to see if we have any PowerShells or Terminals that could save IP connections:

PS C:\users\Oracle12\desktop> Get-ChildItem "HKCU:\Software\Microsoft"

Oh we have a terminal, we'll just plug in "Terminal Server Client" as a subfolder to the same command we just used to view anything left on the terminal:

PS C:\users\Oracle12\desktop> Get-ChildItem "HKCU:\Software\Microsoft\Terminal Server Client"

    Hive: HKEY_CURRENT_USER\Software\Microsoft\Terminal Server Client

Name                           Property
----                           --------
192.168.2.3                    UsernameHint : MyServer\raccoon

There we go, now just copy that IP and we can continue to the next level:
