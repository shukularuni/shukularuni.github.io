Access

SSH: ssh oracle10@oracle.underthewire.tech -p 22

Password: utw_exch9229

Info

The password for oracle11 is the .biz site the user has previously navigated to.

NOTE:
– Don’t include the extension.
– The password will be lowercase no matter how it appears on the screen.

▼ HINT:

The registry is awesome!

Theory

To get the password, as the description says, we have to get name of the site that the user has visited that ends on a `.biz` top-level domain (TLD). For that we just need to view Typed URLs on whatever browser is used on the computer, for those two we can use registry commands Test-Path checks if the path exists and Get-ChildItem gets the stuff inside the registry folders. We don't know the browser, so we can use Test-Path to see if it's either Edge or Internet Explorer, which usually come pre-installed on Windows, then if we see something we'll see the TypedURLs folder inside the browser to check for that `.biz` URL. The commands are like this:

Test-Path "HKCU:\Software\Microsoft\PROGRAM"
Get-ChildItem "HKCU:\Software\Microsoft\PROGRAM\TypedURLs"

Solution

Now that we are inside of the shell, let's check if Edge is installed on the system:

PS C:\users\Oracle10\desktop> Test-Path "HKCU:\Software\Microsoft\Edge"
False

Nope, Edge is not in the system, so let's check for Internet Explorer:

PS C:\users\Oracle10\desktop> Test-Path "HKCU:\Software\Microsoft\Internet Explorer"
True

Yup, now let's go to the TypedURLs folder to view if any of those folders have the `.biz` TLD:

PS C:\users\Oracle10\desktop> Get-Item "HKCU:\Software\Microsoft\Internet Explorer\TypedURLs"

    Hive: HKEY_CURRENT_USER\Software\Microsoft\Internet Explorer\TypedURLs

Name            Property
----            --------
TypedURLs       url1 : http://go.microsoft.com/fwlink/?linkid=66138&clcid=0x409
                url2 : http://google.com
                url3 : http://underthewire.tech
                url4 : http://bimmerfest.com
                url5 : http://nba.com
                url6 : http://yondu.biz
                url7 : http://hardknocks.edu
                url8 : http://installation.org

There we go, now that we have the URL with the `.biz` TLD, we can remove the TLD and http protocol, and we'll get the password:

yondu
https://underthewire.tech/oracle-10
