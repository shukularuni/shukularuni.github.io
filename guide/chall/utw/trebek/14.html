Access

SSH: ssh trebek14@trebek.underthewire.tech -p 22

Password: prindel3003

Info

The password for trebek15 is the output from decoding the PowerShell code found in the account properties of the user account from the previous level PLUS the name of the file on the desktop.

NOTE:
– If the output is “blue_and_red” and the file on the desktop is named “_are_great”, the password would be “blue_and_red_are_great”.
– The password will be lowercase no matter how it appears on the screen.

▼ HINT:

You don’t need any hints!

Theory

To get the password, as the description says, we have to grab the encoded command from previous level and decode it. Yup, that's it, pretty simple, just grab the encoded string from the City property from previous level, the command we used was this Active Directory thing searching for any user that has something in their City property and isn't blank:

(Get-ADUser -Filter * -Properties City).Where({$_.City})

Solution

Now that we are inside of the shell, we can use dir to view the name of the file in the desktop and the command from the previous level to view the City property:

PS C:\users\trebek14\desktop> dir

    Directory: C:\users\trebek14\desktop


Mode                LastWriteTime         Length Name
----                -------------         ------ ----
-a----        8/30/2018  10:49 AM              0 _today

PS C:\users\trebek14\desktop> (Get-ADUser -Filter * -Properties City).Where({$_.City})

City              : agBvAGkAbgBfAHQAaABlAF8AcgBlAGIAZQBsAHMA
DistinguishedName : CN=Prindel,OU=T-65,OU=X-Wing,DC=underthewire,DC=tech
Enabled           : False
GivenName         : Prindel
Name              : Prindel
ObjectClass       : user
ObjectGUID        : 8edb88c2-c69b-4d78-957e-6c1d1b08f2a5
SamAccountName    : Prindel
SID               : S-1-5-21-758131494-606461608-3556270690-2178
Surname           : Prindel
UserPrincipalName : Prindel

So we'll copy the string on the City property and put it into our online converter:

BASE64 DECODE (Base64Decode.com)

INPUT: agBvAGkAbgBfAHQAaABlAF8AcgBlAGIAZQBsAHMA

OUTPUT: jNULoNULiNULnNUL_NULtNULhNULeNUL_NULrNULeNULbNULeNULlNULsNUL

So it seems there is a null character every other character, we can remove them by replacing them with nothing:

BASE64 DECODE (Base64Decode.com) + Find 'NUL' Replace with ''.

INPUT: agBvAGkAbgBfAHQAaABlAF8AcgBlAGIAZQBsAHMA

OUTPUT: join_the_rebels

Perfect, now we just need to join that with the name of the file in the desktop to get the password to the last level:

join_the_rebels_today
