Access

SSH: ssh trebek6@trebek.underthewire.tech -p 22

Password: wlms

Info

The password for trebek7 is the total number of DLLs within the “C:\program files\adobe\” folder and it’s subfolders PLUS the name of the file on the desktop.

NOTE:
– If the count is “9999” and the file on the desktop is named “_abc”, then the password would be “9999_abc”.
– The password will be lowercase no matter how it appears on the screen.

▼ HINT:

When you find the cmdlet to use, you may want to use “Get-Member” to see the additional available properties.

Theory

To get the password, as the description says, all we have to do is count all the DLL files inside the folder given in the description. Yup, that's it, pretty simple right? So for the counting we'll use the Get-ChildItem command to view all the DLLs, then we just count them. Yeah so nothing much to say here, we're just going to display all the files and count them. I don't know why I'm repeating myself so much, it's just that this level is too simple, it's suspiciously simple, whatever just go to the solution part already.

Solution

Now that we are inside of the shell, we can use dir to view the name of the file in the desktop:

PS C:\users\trebek6\desktop> dir

    Directory: C:\users\trebek6\desktop


Mode                LastWriteTime         Length Name
----                -------------         ------ ----
-a----        8/30/2018  10:47 AM              0 _reader

So now we're just going to search recursively through the directory we were given in the description, print any file that has the DLL extension, and count it:

PS C:\users\trebek6\desktop> (Get-ChildItem -Recurse -Filter '*.dll' 'C:\program files\adobe\').Count
40

See? I told you this was suspiciously easy! Now just enter the number and add the name of the file in the desktop:

40_reader
