Info

The password for trebek9 the first 8 bytes of the file located on the desktop. Combine the answer together with NO spaces.

▼ HINT:

https://docs.microsoft.com/en-us/powershell/module/microsoft.powershell.management/get-content?view=powershell-7.1

Theory

To get the password, as the description says, we have to get the first few bytes of a file in the desktop, so basically we have to take the header bytes of that file. Okay, so I don't know how view the bytes of a file in decimal (because that's what the description is implying), I only know how to get the bytes in hexadecimal form, so for this level we'll be using an online converter to pass these hex values over to decimal.

Solution

Now that we are inside of the shell, we can use dir to view the name of the file in the desktop:

PS C:\users\trebek8\desktop> dir

    Directory: C:\users\trebek8\desktop


Mode                LastWriteTime         Length Name
----                -------------         ------ ----
-a----        8/30/2018   5:55 AM         221184 Clone_Trooper_data.pdf

So now we'll use the Format-Hex command to dump the entire file as hexadecimal, and since the file is very long, we'll pipe it to the more command so that it doesn't print the entire 200kb file and basically makes the header bytes inaccessible:

PS C:\users\trebek8\desktop> Format-Hex Clone_Trooper_data.pdf | more

           Path: C:\users\trebek8\desktop\Clone_Trooper_data.pdf

           00 01 02 03 04 05 06 07 08 09 0A 0B 0C 0D 0E 0F

00000000   4D 5A 90 00 03 00 00 00 04 00 00 00 FF FF 00 00  MZ?.............
...

So let's just convert those first 8 characters (the 8 groups of 2 hex chars), and convert them to decimal (no direct hex-decimal, so it's hex-text then text-decimal) to get our password:

HEXADECIMAL TO TEXT TO DECIMAL (online converter)

INPUT: 4D 5A 90 00 03 00 00 00

OUTPUT: 77 90 144 0 3 0 0 0

Now just remove the spaces inbetween and add the name of the file in the desktop:

779014403000
