Access

SSH: ssh trebek3@trebek.underthewire.tech -p 22

Password: droid823

Info

The password for trebek4 is the IP that the user Yoda last logged in from as depicted in the event logs on the desktop PLUS the name of the text file on the user’s desktop.

NOTE:
– If the IP address is “2.3.3.2” and the file on the desktop is named “bobby”, the password would be “2.3.3.2bobby”.

▼ HINT:

https://technet.microsoft.com/en-us/library/ee176852.aspx

Theory

To get the password, as the description says, we have to get the IP address from which a certain "Yoda" last loggen in from. So it seems like this solution is going to be very similar to the one a few levels ago when we needed to search through a security event logs file, though now we're tracking a user login over the network instead of a deleted task. And if I remember correctly the event code for somebody logging in is 4624. So we can attach that to our command from last time, using a filter for the event code and now targeting specifically a user called Yoda:

Get-WinEvent -Path .\WhateverTheFileNameIs.evtx -FilterXPath "*[System[(EventID=4624)]] and *[EventData[Data[@Name='TargetUserName'] and (Data='Yoda')]]"

Solution

Now that we are inside of the shell, we can use dir to view the name of the file in the desktop:

PS C:\users\trebek3\desktop> dir

    Directory: C:\users\trebek3\desktop


Mode                LastWriteTime         Length Name
----                -------------         ------ ----
-a----        8/30/2018  10:46 AM              0 address
-a----        8/30/2018   5:55 AM       99684352 security.evtx

So we have our security event log file and the name of the file in the desktop that goes after the IP address. Now let's put the event log filename in the command from theory and find the instance:

PS C:\users\trebek3\desktop> Get-WinEvent -Path .\security.evtx -FilterXPath "*[System[(EventID=4624)]] and *[EventData[Data[@Name='TargetUserName'] and (Data='Yoda')]]"

   ProviderName: Microsoft-Windows-Security-Auditing

TimeCreated                     Id LevelDisplayName Message
-----------                     -- ---------------- -------
5/11/2017 8:41:10 PM          4624 Information      An account was successfu...
5/11/2017 8:41:10 PM          4624 Information      An account was successfu...
5/11/2017 8:41:08 PM          4624 Information      An account was successfu...
5/11/2017 8:41:06 PM          4624 Information      An account was successfu...

Now let's just check the first one since it's the earliest log in:

PS C:\users\trebek3\desktop> (Get-WinEvent -Path .\security.evtx -FilterXPath "*[System[(EventID=4624)]] and *[EventData[Data[@Name='TargetUserName'] and (Data='Yoda')]]" | Select-Object -First 1).Message
An account was successfully logged on.

Subject:
...

Network Information:
        Workstation Name:       TREBEK
        Source Network Address: 10.30.1.18
        Source Port:            0

...

Perfect, now just take the IP address from Network Information and add in the name of the file in the desktop to get the password:

10.30.1.18address
