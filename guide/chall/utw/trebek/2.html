Access

SSH: ssh trebek2@trebek.underthewire.tech -p 22

Password: mess_cleaner

Info

The password for trebek3 is the name of the executable associated with the C-3PO service PLUS the name of the file on the user’s desktop.

NOTE:
– Don’t include the file extension (i.e.- .exe). If the executable name is “binary.exe” and the file on the desktop is named “1234”, the password would be “binary1234”.
– The password will be lowercase no matter how it appears on the screen.

▼ HINT:

https://technet.microsoft.com/en-us/library/ee176852.aspx

Theory

To get the password, as the description says, we have to get the name of the application (.exe) file that is associated to C-3PO. Normally, we'd use Get-Service for this exact problem, but the thing is, the cmdlet doesn't give us the file path, so for this level it's basically useless. Why then we'll use instead CimInstance, which does give the path associated. And don't forget using the dir command to view the name of the file in the desktop for the other part of the password.

Solution

Now that we are inside of the shell, we can use dir to view the name of the file in the desktop:

PS C:\users\trebek2\desktop> dir

    Directory: C:\users\trebek2\desktop


Mode                LastWriteTime         Length Name
----                -------------         ------ ----
-a----        8/30/2018  10:45 AM              0 823

We can now use the CimInstance command to find any service that mentions C-3PO in any way, and shows all the info for it with the Select-Object command piped:

PS C:\users\trebek2\desktop> Get-CimInstance Win32_Service -Filter 'Name like "C-3PO"' | Select-Object *

Name                    : C-3PO
Status                  : OK
ExitCode                : 0
DesktopInteract         : False
ErrorControl            : Normal
PathName                : g:\star_wars\droid.exe
ServiceType             : Own Process
StartMode               : Auto
Caption                 : C-3PO

Perfect, now just isolate the name of the file from the path that we got from the search and add in the name of the file in the desktop to get the password:

droid823
