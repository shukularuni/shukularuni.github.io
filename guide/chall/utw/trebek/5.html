Access

Password: 01/05/2017_red

Info

The password for trebek6 is the name of the executable that is starting at 3/23/2017 8:08:53 PM via the Software Protection service as depicted in the event log on the desktop.

NOTE:
– Don’t include the file extension (i.e.- .exe).
– The password will be lowercase no matter how it appears on the screen.

Theory

To get the password, as the description says, we have to again, as two of the previous levels, we have to find something in the event log. But now the problem is that it's a little too vague, because what the level is trying to ask us can refer to three different event codes, which are 900, 8200, and 8208. So we can make a command that finds all the events in the event log that have one of the three event codes:

Get-WinEvent -Path .\WhatsTheFileName.evtx -FilterXPath "*[System[(EventID=900)]] or *[System[(EventID=8200)]] or *[System[(EventID=8208)]]"

Solution

Now that we are inside of the shell, we can use dir to view the name of the Windows Event File:

PS C:\users\trebek5\desktop> dir

    Directory: C:\users\trebek5\desktop


Mode                LastWriteTime         Length Name
----                -------------         ------ ----
-a----        8/30/2018   5:55 AM        5312512 application.evtx

Now we can put the name of the event log file into the command from theory and see what we get:

PS C:\users\trebek3\desktop> Get-WinEvent -Path .\application.evtx -FilterXPath "*[System[(EventID=900)]] or *[System[(EventID=8200)]] or *[System[(EventID=8208)]]"

   ProviderName: Microsoft-Windows-Security-SPP

TimeCreated                     Id LevelDisplayName Message
-----------                     -- ---------------- -------
5/11/2017 8:40:06 PM           900 Information      The Software Protection ...
5/11/2017 8:34:45 PM           900 Information      The Software Protection ...
5/11/2017 8:12:11 PM           900 Information      The Software Protection ...
5/17/2017 8:10:00 PM           900 Information      The Software Protection ...
5/17/2017 7:46:03 PM           900 Information      The Software Protection ...
5/17/2017 7:44:11 PM           900 Information      The Software Protection ...
...

Okay, well that was too many entries, we'll now have to filter by time to have a chance at finding this one executable. So let's group the event codes together and make it search for the exact time too:

PS C:\users\trebek3\desktop> Get-WinEvent -Path .\application.evtx -FilterXPath "*[System[(EventID=900 or EventID=16384 or EventID=16394) and TimeCreated[@SystemTime='2017-03-23T20:08:53.000Z']]]"

   ProviderName: Microsoft-Windows-Security-SPP

TimeCreated                     Id LevelDisplayName Message
-----------                     -- ---------------- -------
3/23/2017 8:08:53 PM           900 Information      The Software Protection ...

Perfect, now we only have one entry, and now just add the Message preperty at the end to view the whole message:

PS C:\users\trebek5\desktop> (Get-WinEvent -Path .\application.evtx -FilterXPath "*[System[(EventID=900 or EventID=16384 or EventID=16394) and TimeCreated[@SystemTime='2017-03-23T20:08:53.000Z']]]").Message
The Software Protection service is starting.
Parameters:caller=wlms.exe

Finally! Now just get the name of the executable, remove the extension, and you've got the password:

wlms
