Theory

Just looking at the string and seeing the character set it uses and the double equal sign at the end gives away that this is obviously Base64. So let's decode it.

Solution

Let's put it into CyberChef to decode from Base64:

recipe=From_Base64('A-Za-z0-9%2B/%3D',false,false)  # base64-to-text
