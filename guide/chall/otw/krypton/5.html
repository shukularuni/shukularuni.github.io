Info

FA can break a known key length as well. Lets try one last polyalphabetic cipher, but this time the key length is unknown. Note: the text is writen in American English

Enjoy.

Theory

To get the password, the level description tells us that it is going to be the same as last time, but now we don't know the keylength. Funny thing, that the same website we used in the previous level to decipher the key and decode the password, can also be used to predict the key length. So we'll first need to go to the level files to get the found files and do all this vignere cipher stuff:

cd /krypton/krypton5

Solution

Now that we are logged in the SSH, we can use cd to go to the level folder and see what's there:

~$ cd /krypton/krypton5

/krypton/krypton5$ ls -la
total 28
drwxr-xr-x 2 root     root     4096 Sep 19 07:10 .

-rw-r----- 1 krypton5 krypton5 1915 Sep 19 07:10 found2

-rw-r----- 1 krypton5 krypton5    7 Sep 19 07:10 krypton6

/krypton/krypton5$ cat README
Frequency analysis can break a known key length as well.  Lets try one
last polyalphabetic cipher, but this time the key length is unknown.


Enjoy.

/krypton/krypton5$ cat krypton6
BELOS Z

That's the password we have to decode, now we need to get the size of the key. For that, we'll use the large text from the found files, and like in the last level, all the found files give you the same result. Then we'll put it in the vignere cipher with the "Vigenere Cryptanalysis (Kasiski's Test)" option to get an estimate of the key length:

Turns out the most probable are 3, 6, and 9, you might think 9 is not the correct one because it has one probability less than 3 and 6, but because 3 and 6 are multiples and part of the number 9, it actually makes 9 more probable than the other two because they are multipliers and summators (maybe made the word up but you get the idea). And turns out that the key was in front of our noses all along, being literally "KEYLENGTH". So now the final step is to enter the encoded string from the krypton6 file with the key "KEYLENGTH" and we should have our password:

CIPHERTEXT: BELOS Z

KEY/PASSWORD: KEYLENGTH

RESULT:
RANDO M

Also, remember that the string is divided in blocks, so the real password is "RANDOM". Basically, without the space.

And that's the password! Now we should be good to go to the next level.
