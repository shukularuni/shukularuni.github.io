Theory

To get the solution, as the description says, we have to shift that text however many characters to get the original message, it's no longer just a normal caesar cipher, now it's using the entire ASCII code. So for this challenge, we can't just use an online converter, we'll create a code to go through all the solutions and we'll check one by one to see if there's some real text somewhere.

Solution

We'll create this python code to first convert each character by splitting, from hexadecimal to normal characters, then a loop will go through all 128 possibilities that the characters can swap to, and then print them all with the shift they have at the beginning of each line:
