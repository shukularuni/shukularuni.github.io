Training: GPG

Name: Training: GPG
Tags: Crypto, Training
Score: 2
Description: In this challenge your goal is to setup gpg encryption for your emails.
To do so, generate your keypairs locally and store your public key on wechall.
Then all your mails sent by wechall are encrypted.
To enable GPG encryption, please goto Your account settings.
When you are done, click the button below to send you a mail.

Happy Challenging!

Theory

To get the solution, as the description says, we have to set up our GPG encryption on WeChall, then click the button below to receive a message to decrypt with our GPG password. So to do that, you first need to verify that you have gpg installed (or GnuPG). After that we'll create the password and send it to WeChall.

Solution

So, we'll start to create the password:

~$ gpg --full-generate-key

Here we'll put (1) RSA and RSA to create the correct GPG key:

Your selection? 1
RSA keys may be between 1024 and 4096 bits long.
What keysize do you want? (3072)

Enter a keysize of 4096 because that's the usual thing:

What keysize do you want? (3072) 4096
Requested keysize is 4096 bits
Please specify how long the key should be valid.
         0 = key does not expire
      >n<  = key expires in n days
      >n<w = key expires in n weeks
      >n<m = key expires in n months
      >n<y = key expires in n years
Key is valid for? (0)

Just put 0 or leave blank so that the key never expires, and tell it that everything's correct:

Key is valid for? (0) 0
Key does not expire at all
Is this correct? (y/N) y

GnuPG needs to construct a user ID to identify your key.

Here you can put any name, I put my username. Then it's going to ask for email address, this one has to be the address you use for WeChall, then in the comment, you don't need to enter anything:

Press `O` for Okay, now finally, it will ask for a password, make a great password that you can remember, as you will be using it later into the challenge, when you enter the password you can move the mouse around a little to create some randomness so that the GPG key has better security:
