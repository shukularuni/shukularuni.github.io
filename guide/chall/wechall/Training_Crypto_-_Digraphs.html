Theory

To get the solution, as the description says, we have to decode this matrix cipher, but because we have no idea what it says, I think we can do it in a really creative way. In the previous level of substitution, we used a tool to find letter density and get the original message back. So because all this is just letters swapped for two-letter-blocks, we can just assign each combination a letter and put it on that substitution solver, so that we can get the original message.

Solution

First, let's separate all the two-letter-blocks, the parentheses are spaces:

Now that we have each letter separated, we can replace each two-letter-block with a single placeholder letter, treating every unique block as if it were a symbol in the substitution cipher. The specific letters we choose don't matter; what matters is that the same block always gets the same letter, and no two different blocks share a letter, and spaces are kept the same so word boundaries are still there. So at the end, the resulting text can be run through the substitution solver, which will work based on the letter frequencies rather than the actual placeholder letters. So in my case, I just went through each two-letter-block assigning it in alphabetical order, because yes:
