Theory

To get the solution, as the description says, we have to decode this weird string. Because they mentioned substitution ciphers, and the challenge is literally named Substitution 1, we'll obviously have to decode this with via a substitution cipher. Usually this type of cipher is solved by looking at the letter density and guessing which swapped letters are which, but I do know a little website that I've used for some time now that is really good at density mapping letters of the alphabet on their Substitution Solver page. So we'll copy the given string and put it in the page.

Solution

So let's check what the message might be with the solver:

Substitution Solver (guballa.de)

Input: RW HSF TGNJLSHW LEX WEA PTI YFTX HSJM NW QYJFIX J TN JNOYFMMFX DFYW CFGG XEIF WEAY MEGAHJEI BFW JM QIQYSFYYYMQI HSJM GJHHGF PSTGGFILF CTM IEH HEE STYX CTM JH

abcdefghijklmnopqrstuvwxyz
ukwvoeltniqgsmpcfbhaxjydrz

Result: BY THE ALMIGHTY GOD YOU CAN READ THIS MY FRIEND I AM IMPRESSED VERY WELL DONE YOUR SOLUTION KEY IS FNFRHERRRSFN THIS LITTLE CHALLENGE WAS NOT TOO HARD WAS IT

There we go! That's the solution.
