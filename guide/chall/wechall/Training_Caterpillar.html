(caterpillar.png)[My perfectly drawn image of a caterpillar]

Happy Challenging!

Theory

To get the solution, as the description says, we have to find some hidden text in the image, and judging by how it's conveniently drawn in similarly colored balls that have just random color patterns, I think it's not so far off guessing that the hex values of the colors might be the message itself. I also did check for any metadata, but there's nothing there, clarifying our other theory even more.

Solution

By how different the hue of the head ball, I don't think that last one is part of the puzzle. So first let's get the hex values from each ball in the image. :

#87c229, #79ba1e, #3e5212, #43520e, #b0c229, #557315, #7fb031, #aad442

Yes, I went through each one individually getting the value, so now let's try Hexadecimal decoding on these values:

HEXADECIMAL TO TEXT (online converter)

INPUT: 87c22979ba1e3e521243520eb0c2295573157fb031aad442

OUTPUT: .A)y*.>R.CR.*A)Us..*1*OB

Huh, that doesn't seem human readable. Although the theory still stands strong as I haven't found any other way of getting a hidden message yet. Hex is the most popular option, but we can use others like HSB, CMYK, or Lab. I'm not going to convert the Hex values to HSB, because that's harder than just getting the HSB values from the image itself again. So again I went through each ball getting the HSB value:

hsl(83, 79, 76), hsl(85, 84, 73), hsl(79, 78, 32), hsl(73, 83, 32), hsl(67, 79, 76), hsl(79, 82, 45), hsl(83, 72, 69), hsl(77, 69, 83)

Now that definetely looks like readable text, although HSB values are in decimal, not hexadecimal:

DECIMAL TO TEXT (online converter)

INPUT: 83, 79, 76, 85, 84, 73, 79, 78, 32, 73, 83, 32, 67, 79, 76, 79, 82, 45, 83, 72, 69, 77, 69, 83

OUTPUT: SOLUTION IS COLOR-SHEMES

There we go! That's the solution.
