Theory

To complete this challenge, we have to look through the PHP source code and find a way to enter into the admin account with the login below. So, I'll check the code and will see you in a moment.

Solution

Okay, so looking through the code this is a really simple classic SQL injection disguised with a lot of spacing to make it look more complex. So, if we go over to line 43, we can take a look at this goldmine of glitches:

$query = "SELECT * FROM users WHERE username='$username' AND password='$password'";

With this on the line, we can just do a simple SQL injection to comment out after the admin username so that it doesn't even process the password and lets us get into admin. The password can be anything since it will be commented out:

Username: admin' --
Password:

Auth1: That doesn't seem right.

Huh, that didn't work, we can try to comment with the hashtag symbol instead, this usually happens with testing environments like WeChall:

Username: admin' #
Password: anything

Auth1: Welcome back, %1$s!

Little weird that we are refered as some strange characters. There we go! It just gets marked as complete when doing that.
