Theory

To get the solution, as the description says, we have to decode this weird kind of binary with the program they're giving us. So let's download it and see how we can get this to be normal ASCII text.

Solution

Well I don't know how to use the program, so I'm gonna go to the help tab:

Although, while messing with the BitsPerBlock thing, I discovered that the binary they gave us is 7 bits. So basically instead of the usual binary being eight consecutive 0s and 1s, it's just seven. So now I can make this little code to divide these in 7 so that we can convert it back to ASCII:

>>> thing = "theBinaryTheyGaveUs".replace(" ",'')

>>> chunks = [thing[i:i+7] for i in range(0, len(thing), 7)]

>>> txt = ''

>>> for i in chunks:
...   txt += i + ', '

>>> txt
'1010100, 1101000, 1101001, 1110011, 0100000, 1110100, 1100101, 1111000, 1110100, 0100000, 1101001, 1110011, 0100000, 0110111, 0101101, 1100010, 1101001, 1110100, 0100000, 1100101, 1101110, 1100011, 1101111, 1100100, 1100101, 1100100, 0100000, 1100001, 1110011, 1100011, 1101001, 1101001, 0101110, 0100000, 1011001, 1101111, 1110101, 1110010, 0100000, 1110000, 1100001, 1110011, 1110011, 1110111, 1101111, 1110010, 1100100, 0100000, 1101001, 1110011, 0100000, 1100101, 1100001, 1110011, 1111001, 1110011, 1110100, 1100001, 1110010, 1110100, 1100101, 1110010, 0101110, '
