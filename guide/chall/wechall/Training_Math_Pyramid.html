Theory

To get the solution, as the description says, we'll have to do some math to solve this challenge. We have a square-based pyramid where every base edge has length `a`, every side (slanted) edge also has length `a`, so total edges = 8 meaning they're all equal. And that forces the apex to be directly above the center of the square, otherwise some edges would be longer. So let's try solving a formula for this pyramid.
