Theory

To get the solution, as the description says, we have to get a message, although that message we get is a time bomb, and it blows itself up in like a single second. So unless you're like The Flash or something, you can't do this manually, but you know who can do this? A program, just a code that gets the message and opens it in the other link automatically. I'm going to use python because it's probably the easiest approach. So yeah here's a code, it's pretty simple, first we get the content of the link we get the message from with the cookie of your profile found in the only cookie of your session in any WeChall.net webpage, then immediately get the other link with the message there, and get the HTML of the result just in case:

msg = requests.get("https://www.wechall.net/en/challenge/training/programming1/index.php?action=request", cookies={"WC": "<redacted>"})

req = requests.get(f"https://www.wechall.net/en/challenge/training/programming1/index.php?answer={msg.text}", cookies={"WC": "<redacted>"})

print(req.text)
main.py
