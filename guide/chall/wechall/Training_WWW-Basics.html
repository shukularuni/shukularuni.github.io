Training: WWW-Basics

Name: Training: WWW-Basics
Tags: HTTP, Training
Score: 2
Description: As you know, a popular part of the internet is called the World-Wide-Web.
The www consists of http servers, which serve html documents and other content types.
Your mission here is easy. Setup a www server on your computer, make it accessible by the outside, and serve the file http://<redacted>:80/shukularuni/shukularuni.html.
The content of the file has to be exactly:
My name is shukularuni and iChall.
The content has to be exact 34 bytes in length.

As soon as you have set it up you can choose the port your server is using, 80 by default, and hit the button.

Enjoy!

Theory

I am broke, so I wont be getting the solution, but I can tell you how YOU can do it. It's really simple, WeChall just wants to verify that you can host a publicly reachable HTTP server and serve a file with exact contents at the specific URL. It checks the page by making an HTTP request to the IP address you're logged in from, so you won't be able to do that from your own computer, since it's buried behind a NAT connection, so anything you host there will not reach outside of your own home WiFi. So to host a publicly reachable web server from a personal machine is basically not possible (unless you port-forward which I won't tell you how to do because it's kinda hard and annoying).

So the strategy we'll use is to use a VPS with a public IP, a minimal server for the lowest cost because this challenge only requires serving a single static file with no more than 40 bytes of bandwidth. Considering that you don't want to waste that much money on a few worthless internet points, we'll stick to Vultr, which is cheap and well-documented (good documentation so that I can know how to set up the server without buying a server myself).

So because we're going to host a webpage on a server, the server doesn't have like a browser or is some kind of visual system, it's just a normal terminal you control, so we'll have to create the webpage and send it to WeChall all through terminal commands or scripts.

Solution

First, create a Vultr account, deposit the minimum, and pick the cheapest cloud compute instance. Then put the details of the server, the location, server type (Application - LEMP - CentOS 7), the smallest plan ($5/mo tier), and deploy. After deploying you'll see a public IPv4 address,you can visit this IP as an address to view a default server webpage.

Now SSH into your VPS server with the IP address and the password you got:

~$ ssh root@<YOUR SERVER IP HERE>

Once inside, go to nginx root, or well, where the files your website serves when someone enters the IP address:

cd /usr/share/nginx/html

Create the folder with your username and enter into it:

mkdir shukularuni
cd shukularuni

Finally create the HTML file with the content that WeChall gave to you:

echo -n "My name is shukularuni and iChall." > shukularuni.html

Now verify that the message is in the web server with the given IP:

http://<YOUR SERVER IP HERE>/shukularuni/shukularuni.html

Now install Python with yum, then get the requests library so that we can send the signal to WeChall that we have the server:

yum -y install python-pip
python -m pip install --upgrade pip
python -m pip install requests

Log into WeChall, and on any page of its website, go to the cookies, and grab the value from the cookie called "WC", this is your secret user ID (don't share it), which can be used to be logged in your user and to do challenges from other local entry points, like the server you're setting up:

= Cookies =

WC  XXXXXXXX-XXXXX-XXXXXXXXXXXXXXXX  wechall.net  /  33  ....  ....  ..........

Now write this Python code to a file in the home folder with your WC cookie on the redacted part:

import requests
u='https://www.wechall.net/challenge/training/www/basic/index.php'
c={'WC':'<redacted>'}
a={'port':'80','go':'I have set it up. Please check my server.'}
r=requests.post(u,cookies=c,data=a)
print r.text

Only thing left is to run the file with Python and you should get the response with the points won:

~$ python code.py
