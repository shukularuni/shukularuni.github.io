Theory

To get the solution, as the description says, we have to decode this giant block of text. And as it is probably so painfully obvious, is that there's something to do with the caps on the letters, because the Bacon Cipher is mentioned in the description and is part of the title. So I think a simple python code should help us get, for example, A's from uppercase letters, and B's from lowercase, and just skip whatever isn't a letter:

''.join('A' if i.isalpha() and i.isupper() else ('B' if i.isalpha() and i.islower() else '') for i in txt)

Solution
