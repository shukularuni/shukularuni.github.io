StegoRSA

Name: StegoRSA
Description: A message has been encrypted using RSA. The public key is gone… but someone might have been careless with the private key. Can you recover it and decrypt the message? Download the flag and image.
Author: Yahaya Meddy
Tags: Easy, Cryptography, picoCTF 2026
Challenge from: picoCTF 2026
Files: flag.enc, image.jpg
Hints:
1. Metadata can tell you more than you expect.
2. Hex can be turned back into a key file.

Theory

According to the description, to get the flag it seems we have to decrypt some RSA message with a private key that should be able to be found somewhere, since somebody forgot to be careful with it. Seems pretty simple, first we need to find a key in some way and then use it on the encrypted flag. We don't know much else about the challenge until this point but it seems weird they're giving us an "image", seems too random to be accidental, it might contain the accidentally leaked private key we were told about on the description, although this is all speculation so let's just go on with it.

Solution

So first let's download the files and see what they are to get a better idea of the challenge:

shukularuni@SHUKULAP:~$ file flag.enc

shukularuni@SHUKULAP:~$ exiftool image.jpg

Well that's a surprise, the comment of the image seems to have some interesting, let's use CyberChef to see what this comment actually says:

HEX TO TEXT (CyberChef)

INPUT: 2d2d2d2d2d424547494e2050524956415445204b45592d...

OUTPUT:
-----BEGIN PRIVATE KEY-----
MIIEvgIBADANBgkqhkiG9w0BAQEFAASCBKgwggSkAgEAAoIBAQCyi2qh7k1+l1Q7
xkVIM6mZY9swyl9433vHYRYMrBEvH+/DZws2stMKlFY9KgJbdZjx0qcgm2JQ6Tqp
...

Well that seems to be our private key, now to finish the challenge is pretty easy, since all we have to do is use something like OpenSSL to decrypt the encrypted flag to text by using this key. So just add this key that's been converted from hex from the image comment to a file and paste the exact output you get into nano or vim, in my example it's key.pem, remember the pem extension for key files, and file to put the decrypted flag:

shukularuni@SHUKULAP:~$ nano key.pem
-----BEGIN PRIVATE KEY-----
MIIEvgIBADANBgkqhkiG9w0BAQEFAASCBKgwggSkAgEAAoIBAQCyi2qh7k1+l1Q7
xkVIM6mZY9swyl9433vHYRYMrBEvH+/DZws2stMKlFY9KgJbdZjx0qcgm2JQ6Tqp
...

shukularuni@SHUKULAP:~$ openssl pkeyutl -decrypt -inkey key.pem -in flag.enc -out flag.txt

shukularuni@SHUKULAP:~$ cat flag.txt
picoCTF{rs4_k3y_1n_1mg_a9a7c4c9}

There we go! That's the flag.
