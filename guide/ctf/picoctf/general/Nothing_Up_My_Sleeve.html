Nothing Up My Sleeve

Name: Nothing Up My Sleeve
Description: Let's check that your internet connection is working. This flag is 'in-the-clear', I promise! Download flag.txt
Author: Q0h313th
Tags: Medium, General Skills, picoCTF 2020 Mini-Competition
Challenge from: picoCTF 2020 Mini-Competition
Files: flag.txt
Hints:
1. If all you had access to was a shell, you could use wget to download the file at the URL above!

Theory

According to the description, to get the flag we have to download the file. So I have a question... is that it? It's weird that this challenge is rated as Medium if it's literally just checking that you're not blind and have an internet connection, which you should have if you're able to see the freaking challenge. What an odd one huh, we'll at least download the file with wget to give the challenge some credit, by the way I'm not against the challenge, if it was rated as Easy I'd let it pass, I'm just kinda confused.

Solution

We'll first download the file linked on the description of the challenge:
