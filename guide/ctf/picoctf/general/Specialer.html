Specialer

Name: Specialer
Description: Reception of Special has been cool to say the least. That's why we made an exclusive version of Special, called Secure Comprehensive Interface for Affecting Linux Empirically Rad, or just 'Specialer'. With Specialer, we really tried to remove the distractions from using a shell. Yes, we took out spell checker because of everybody's complaining. But we think you will be excited about our new, reduced feature set for keeping you focused on what needs it the most. Please start an instance to test your very own copy of Specialer. ssh -p 61315 ctf-player@saturn.picoctf.net. The password is d8819d45
Author: LT 'syreal' Jones
Tags: Medium, General Skills, picoCTF 2023, bash, ssh
Challenge from: picoCTF 2023
Hints:
1. What programs do you have access to?

Theory

According to the description, to get the flag we have to enter into a specialer shell, which now doesn't make commands be properly spelled and capitalized, which was an odd feature, but this new one just doesn't have as many commands?? Alright, we'll see what we can do in there. We'll enter into the SSH on the solution part.

Solution

We'll connect to the SSH with the command we are given in the challenge description:

shukularuni@SHUKULAP:~$ ssh -p 61315 ctf-player@saturn.picoctf.net
ctf-player@saturn.picoctf.net's password: d8819d45

Specialer$ ls
-bash: ls: command not found

Specialer$ cat flag.txt
-bash: cat: command not found

Yup, we basically don't have most of the useful commands, although I think there was another way to both list files and print them, with echo:

Specialer$ echo *
abra ala sim

I don't know if those are files or folders, so we'll use the echo replacement to cat:

Specialer$ echo "$(<abra)"
Specialer$ echo "$(<ala)"
Specialer$ echo "$(<sim)"

Yeah, they don't have anything because they're folders. So we'll use these two echo commands to look through the three directories and print any files that they might have:

Specialer$ cd sim

Specialer$ echo *
city.txt salabim.txt

Specialer$ echo "$(<city.txt)"
05ed181c-4aa0-4d4a-8505-2fe6ca9097d3

Specialer$ echo "$(<salabim.txt)"
#He was so kind, such a gentleman tied to the oceanside#

Specialer$ cd ~/abra

Specialer$ echo *
cadabra.txt cadaniel.txt

Specialer$ echo "$(<cadabra.txt)"
Nothing up my sleeve!

Specialer$ echo "$(<cadaniel.txt)"
Yes, I did it! I really did it! I'm a true wizard!

Specialer$ cd ~/ala

Specialer$ echo *
kazam.txt mode.txt

Specialer$ echo "$(<kazam.txt)"
return 0 picoCTF{y0u_d0n7_4ppr3c1473_wh47_w3r3_d01ng_h3r3_c42168d9}

Specialer$ echo "$(<mode.txt)"
Yummy! Ice cream!

There we go! That's the flag.
