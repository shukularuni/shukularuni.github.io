Log Hunt

Name: Log Hunt
Description: Our server seems to be leaking pieces of a secret flag in its logs. The parts are scattered and sometimes repeated. Can you reconstruct the original flag? Download the logs and figure out the full flag from the fragments.
Author: Yahaya Meddy
Tags: Easy, General Skills, picoMini by CMU-Africa, browser_webshell_solvable
Challenge from: picoMini by CMU-Africa
Files: server.log
Hints:
1. You can use grep to filter only matching lines from the log.
2. Some lines are duplicates; ignore extra occurrences.

Theory

According to the description, to get the flag we have to download the server logs, then search and reconstruct all the parts of the flag, it seems pretty simple. So let's go to the terminal to download the file and check what it has in it.

Solution

First, we'll download the file to see what it might be:
