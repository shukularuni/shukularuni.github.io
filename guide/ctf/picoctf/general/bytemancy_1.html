bytemancy 1

Name: bytemancy 1
Description: Can you conjure the right bytes? The program's source code can be downloaded here. Connect to the program with netcat: $ nc foggy-cliff.picoctf.net 49440
Author: LT 'syreal' Jones
Tags: Easy, General Skills, picoCTF 2026
Challenge from: picoCTF 2026
Files: app.py
Hints:
1. No copy-pasta, please - use Python!

Theory

According to the description, to get the flag we have to do some byte games inside a NetCat, so nothing else it seems that the description tells us, so let's continue into the solution and see inside the NetCat connection.

Solution

We'll enter with the NetCat command we were given on the description:

shukularuni@SHUKULAP:~$ nc foggy-cliff.picoctf.net 49440
&------[ BYTEMANCY-1 ]------&
8&0|)BVA*8&0|)BAA*8&0|)BAA*8&

Send me ASCII DECIMAL 101 1751 times, side-by-side, no space.

8&0|)BVA*8&0|)BAA*8&0|)BAA*8&
&-------------*-------------&
==>

So it seems that this is the same as the previous level, bytemancy 0, although this time instead of entering 3 e's, now it's over 1700, we know 101 is the decimal for the letter e from converting it the previous level. Although we can pipe through to the NetCat that many e's with python, by printing 1751 e's from right there on the terminal:

print('e'*1751)

python3 -c "print('e'*1751)"

So now we just need to attach the python code on the terminal to the NetCat so that it pipes all the e's and we get the flag:

shukularuni@SHUKULAP:~$ nc foggy-cliff.picoctf.net 49440
&------[ BYTEMANCY-1 ]------&
8&0|)BVA*8&0|)BAA*8&0|)BAA*8&

Send me ASCII DECIMAL 101 1751 times, side-by-side, no space.

8&0|)BVA*8&0|)BAA*8&0|)BAA*8&
&-------------*-------------&
==> picoCTF{h0w_m4ny_e's???_b6277f00}

^C

There we go! That's the flag.
