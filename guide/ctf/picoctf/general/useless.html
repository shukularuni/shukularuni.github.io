useless

Name: useless
Description: There's an interesting script in the user's home directory The work computer is running SSH. We've been given a script which performs some basic calculations, explore the script and find a flag. Hostname: saturn.picoctf.net, Port: 57464, Username: picoplayer, Password: password.
Author: Loic Shema
Tags: Medium, General Skills, picoCTF 2023, man
Challenge from: picoCTF 2023

Theory

According to the description, to get the flag we have to look through an "interesting" script inside the SSH shell, there's nothing much else to be discussed about that, all we know for now. So we'll enter into that SSH with the ssh command of course, with the user [at](@) the host, then add the port through the p option, and enter the password when you're asked later:

ssh picoplayer@saturn.picoctf.net -p 57464

Solution

So we'll enter the SSH machine with the command from theory and take a look around the home directory:

shukularuni@SHUKULAP:~$ ssh picoplayer@saturn.picoctf.net -p 57464
picoplayer@saturn.picoctf.net's password: password

picoplayer@challenge:~$ ls
useless

So there's a single file, the script mentioned, and it can be ran, so let's run it and see what happens:

picoplayer@challenge:~$ ./useless
Read the code first

Oh, we need to read the code first to understand how the program actually works, since the program's not compiled we can just use cat to view the whole code of this program:

picoplayer@challenge:~$ cat useless
#!/bin/bash
# Basic mathematical operations via command-line arguments

if [ $# != 3 ]
then
  echo "Read the code first"
else
  if [[ "$1" == "add" ]]
  then
    sum=$(( $2 + $3 ))
    echo "The Sum is: $sum"

  elif [[ "$1" == "sub" ]]
  then
    sub=$(( $2 - $3 ))
    echo "The Substract is: $sub"

  elif [[ "$1" == "div" ]]
  then
    div=$(( $2 / $3 ))
    echo "The quotient is: $div"

  elif [[ "$1" == "mul" ]]
  then
    mul=$(( $2 * $3 ))
    echo "The product is: $mul"

  else
    echo "Read the manual"

  fi
fi

And look over there on the end, it tells us to read the manual. So we can open the manual of this program by using the command man, and maybe it can help us understand better this program so that we can get the flag:

picoplayer@challenge:~$ man useless
useless
     useless, — This is a simple calculator script

SYNOPSIS
     useless, [add sub mul div] number1 number2

DESCRIPTION
     Use the useless, macro to make simple calulations like addition,subtraction, multiplication and division.

Examples
     ./useless add 1 2
       This will add 1 and 2 and return 3

     ./useless mul 2 3
       This will return 6 as a product of 2 and 3

     ./useless div 6 3
       This will return 2 as a quotient of 6 and 3

     ./useless sub 6 5
       This will return 1 as a remainder of substraction of 5 from 6

Authors
     This script was designed and developed by Cylab Africa

     picoCTF{us3l3ss_ch4ll3ng3_3xpl0it3d_6194}

There we go! That's the flag.
