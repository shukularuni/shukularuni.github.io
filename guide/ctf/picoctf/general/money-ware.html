money-ware

Name: money-ware
Description: Flag format: picoCTF{Malwarename} The first letter of the malware name should be capitalized and the rest lowercase. Your friend just got hacked and has been asked to pay some bitcoins to 1Mz7153HMuxXTuR2R1t78mGSdzaAtNbBWX. He doesn't seem to understand what is going on and asks you for advice. Can you identify what malware he's being a victim of?
Author: Juni19
Tags: Easy, General Skills, picoCTF 2023, osint, Retired
Challenge from: picoCTF 2023
Hints:
1. Some crypto-currencies abuse databases exist; check them out!
2. Maybe Google might help.

Theory

According to the description, to get the flag we have to search for the malware name with that bitcoin address. This is a very simple challenge, we can use Google to search for specifically that bitcoin address, and maybe we can find some databases or journals that document the malware, from there grab the name of this mysterious malware.

Solution

So we'll just Google the address followed by malware, which directs us to this search webpage:

https://google.com/search?q=1Mz7153HMuxXTuR2R1t78mGSdzaAtNbBWX+malware

That must be the malware name, although there are two names NotPetya and Petya, we'll try to submit both names and see which one is the correct answer:

Submit Flag: picoCTF{NotPetya}

Check: Sorry, that flag is incorrect!
Submit Flag: picoCTF{Petya}

Check: Hurray! You solved this challenge.

There we go! That's the flag.
