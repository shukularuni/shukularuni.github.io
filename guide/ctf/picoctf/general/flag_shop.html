flag_shop

Name: flag_shop
Description: There's a flag shop selling stuff, can you buy a flag? Source. Connect with nc fickle-tempest.picoctf.net 61854.
Author: Geoffrey Njogu
Tags: Medium, General Skills, picoCTF 2019
Challenge from: picoCTF 2019
Files: store.c
Hints:
1. Two's compliment can do some weird things when numbers get really big!

Theory

According to the description, to get the flag we have to go through a flag shop inside a NetCat. Not much more information for now, so we'll skip to the solution.

Solution

First we'll enter into the NetCat from the description of the challenge:

shukularuni@SHUKULAP:~$ nc fickle-tempest.picoctf.net 61854
Welcome to the flag exchange
We sell flags

1. Check Account Balance

2. Buy Flags

3. Exit

There are two flags here, knockoff flags that only cost 900, and the real flag that costs 100k, so let's check the code and see if we can find anything interesting:

int number_flags = 0;
scanf("%d", &number_flags);
if(number_flags>  0){
    int total_cost = 0;
    total_cost = 900*number_flags;
    printf("\nThe final cost is: %d\n", total_cost);
    if(total_cost <= account_balance){
        account_balance = account_balance - total_cost;

So if you look closely at this part of the code, we can see an integer overflow on the calculation of the cost of all knockoff flags. What we can do here is make the program calculate the cost over the 32-bit limit (2147483647), so that it flips around and instead gives us an inmense amount of money. To get the program to go over the limit we'd have to buy at least 2.4 million knockoff flags, since 2400000*900 is already well over the 32-bit integer limit, so we can enter that as our number:

Currently for sale
1. Defintely not the flag Flag
2. 1337 Flag
1
These knockoff Flags cost 900 each, enter desired quantity
2400000

The final cost is: -2134967296

Your current balance after transaction: 2134968396

Perfect! Now that we purchased so many flags the number flips around and instead gives us money, we can finally buy the real flag (the 1337 flag):
