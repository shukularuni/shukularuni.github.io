bytemancy 2

Name: bytemancy 2
Description: Can you conjure the right bytes? The program's source code can be downloaded here. Connect to the program with netcat: $ nc lonely-island.picoctf.net 54172
Author: LT 'syreal' Jones
Tags: Medium, General Skills, picoCTF 2026
Challenge from: picoCTF 2026
Files: app.py
Hints:
1. There's no way to print these bytes
2. Use pwntools to send raw bytes over the network

Theory

According to the description, to get the flag we have to do some byte games inside a NetCat, so nothing else it seems that the description tells us, so let's continue into the solution and see inside the NetCat connection.

Solution

We'll enter with the NetCat command we were given on the description:

shukularuni@SHUKULAP:~$ nc lonely-island.picoctf.net 54172
&------[ BYTEMANCY-2 ]------&
8&0|)BVA*8&0|)BAA*8&0|)BAA*8&

Send me the HEX BYTE 0xFF 3 times, side-by-side, no space.

8&0|)BVA*8&0|)BAA*8&0|)BAA*8&
&-------------*-------------&
==>

Still the same stuff as the other two challenges, although this time we are dealing with weird bytes. It's still possible to do from python but the command might be a little too long I think, so instead I'll use the linux command printf to print characters FF and pipe them into the NetCat:

shukularuni@SHUKULAP:~$ printf '\xff\xff\xff' | nc lonely-island.picoctf.net 54172
&------[ BYTEMANCY-2 ]------&
8&0|)BVA*8&0|)BAA*8&0|)BAA*8&

Send me the HEX BYTE 0xFF 3 times, side-by-side, no space.

8&0|)BVA*8&0|)BAA*8&0|)BAA*8&
&-------------*-------------&
==>

Well that's odd, it didn't give anything and it's still waiting for an input. Oh wait! I forgot to click enter to enter the answer into the NetCat, since printf doesn't add a new line character after the content we have to add it ourselves so it actually gets inputted into the app and then give us the flag:

shukularuni@SHUKULAP:~$ printf '\xff\xff\xff\n' | nc lonely-island.picoctf.net 54172
&------[ BYTEMANCY-2 ]------&
8&0|)BVA*8&0|)BAA*8&0|)BAA*8&

Send me the HEX BYTE 0xFF 3 times, side-by-side, no space.

8&0|)BVA*8&0|)BAA*8&0|)BAA*8&
&-------------*-------------&
==> picoCTF{3ff5_4_d4yz_86a39d4b}

^C

There we go! That's the flag.
