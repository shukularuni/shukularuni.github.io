Theory

According to the description, to get the flag we have to make the service fail, for that we'll need to know like what stuff is written on that HAProxy configuration file and see the app itself. So we'll skip over to the solution part to take a look at the super secure website and the files that we are given.

Solution

Okay I have looked at the files, and it seems that the server has a backup, which reveals the flag:

IS_BACKUP = yes

So all we need to do is send requests until the server returns a 503 error, looking at the files it seems that the error is gotten at around the 300 requests in a minute mark. So for this we can create a python code that gets the webpage as many times as possible until it gives the 503 error, where it stops sending requests, also I added a little counter that tells us how many requests have been done every 50 requests to know how it's going:

import requests

for i in range(1,600):
  res = requests.get("http://mysterious-sea.picoctf.net:65169/")
  if res.status_code == 503:
    break
  elif i%50 == 0:
    print(i)

So if we run this python code, it should send requests until the server gives up (if you run the code and takes too long, try running it on your webshell instead cause it goes faster there):

shukularuni@SHUKULAP:~$ python3 failure.py
50
100
150
200
250
300

Perfect! The program stopped at 300 requests, now quickly go to the webpage on your browser, or reload the page to see if we got the flag:

picoCTF{f41l0v3r_f0r_7h3_w1n_a73e52a4}

There we go! That's the flag.
