Based

Name: Based
Description: To get truly 1337, you must understand different data encodings, such as hexadecimal or binary. Can you get the flag from this program to prove you are on the way to becoming 1337? Connect with nc fickle-tempest.picoctf.net 59923.
Author: Alex Fulton / Daniel Tunitis
Tags: Medium, General Skills, picoCTF 2019
Challenge from: picoCTF 2019
Hints:
1. I hear python can convert things.
2. It might help to have multiple windows open.

Theory

According to the description, to get the flag we will be looking at different encodings like hexadecimal and binary, which are really easy to distinguish from other encodings, hexadecimal uses all the numbers plus the first 6 letters of the alphabet (ABCDEF), and binary only uses 0 and 1. And we can decode them pretty easily with CyberChef, an online converter, and it's also very fast. So let's go to the solution part to enter into this NetCat and see what the challenge is about.

Solution

First we'll enter into the NetCat with the command we are given on the description of the challenge:

shukularuni@SHUKULAP:~$ nc fickle-tempest.picoctf.net 56301
Let us see how data is stored
socket
Please give the 01110011 01101111 01100011 01101011 01100101 01110100 as a word.
...
you have 45 seconds.....

Input:

Oh it seems we need to be quick for this challenge, so I hope you already had CyberChef open behind your terminal because we can't lose a second here. So in CyberChef we'll grab the From Binary block and paste the binary that the NetCat just gave us and that will give us the input:

recipe=From_Binary('Space',8)  # binary-to-text

- INPUT: 01110011 01101111 01100011 01101011 01100101 01110100

- OUTPUT:
socket
Input:
socket
Please give me the o146 o141 o154 o143 o157 o156 as a word.
Input:

This seems to be Octal encoding, from the really high three-digit numbers and the O before each number, that we have to remove quickly because octal doesn't usually have any letters next to them:

recipe=From_Octal('Space')  # octal-to-text

- INPUT: 146 141 154 143 157 156

- OUTPUT:
falcon
Input:
falcon
Please give me the 6c696d65 as a word.
Input:

That's hexadecimal, because of the alphanumeric string. So quickly go back to CyberChef, paste that string, and put the From Hex block:

recipe=From_Hex('Auto')  # hex-to-text

- INPUT: 6c696d65

- OUTPUT:
lime
Input:
lime
You've beaten the challenge
Flag: picoCTF{learning_about_converting_values_aeBEA593}
^C

There we go! That's the flag.
