mus1c

Name: mus1c
Description: I wrote you a song. Put it in the picoCTF{} flag format.
Author: Danny
Tags: Medium, General Skills, picoCTF 2019
Challenge from: picoCTF 2019
Files: lyrics.txt
Hints:
1. Do you think you can master rockstar?

Theory

According to the description, to get the flag we have to take a look at a song? This is an odd request, but alright. So we'll check out this file on the solution part to see if there might be any hidden code or code word that might be the flag.

Solution

Okay, so looking at this lyrics file I have this thought of having seen something like this before, because if I remember correctly there's an esoteric programming language called Rockstar that literally relies on the length of words to write code or hide messages. So we'll open this website I found that compiles Rockstar online:

https://codewithrockstar.com/online.html
-=- INPUT -=-
Pico's a CTFFFFFFF
my mind is waitin
It's waitin

...

build it up, up
shout it
shout Pico

And if we hit the run button, in this case it's called the Rock button, we get this output:

-=- OUTPUT -=-
114
114
114
111
99
107
110
114
110
48
49
49
51
114

As you can see it's a bunch of numbers, but this is decimal encoding. We know this because letters in decimal are in the range between 90 and 120, and we know this isn't hexadecimal cause every character has a fixed length of two numbers or letters, on this one we can see three-digit numbers like 107. So to decode this decimal we'll use an online decoding tool such as CyberChef:

recipe=From_Decimal('Space',true)  # decimal-to-text

- INPUT: 114 114 114 111 99 107 110 114 110 48 49 49 51 114

- OUTPUT:
rrrocknrn0113r

Perfect, that must be the code! Now we'll insert that into the picoCTF flag format to get this flag as a result:

picoCTF{rrrocknrn0113r}

There we go! That's the flag.
