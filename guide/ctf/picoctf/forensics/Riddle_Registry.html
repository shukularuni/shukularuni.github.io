Riddle Registry

Name: Riddle Registry
Description: Hi, intrepid investigator! 📄🔍 You've stumbled upon a peculiar PDF filled with what seems like nothing more than garbled nonsense. But beware! Not everything is as it appears. Amidst the chaos lies a hidden treasure—an elusive flag waiting to be uncovered. Find the PDF file here Hidden Confidential Document and uncover the flag within the metadata.
Author: Prince Niyonshuti N.
Tags: Easy, Forensics, picoMini by CMU-Africa, browser_webshell_solvable
Challenge from: picoMini by CMU-Africa
Files: confidential.pdf
Hints:
1. Don't be fooled by the visible text; it's just a decoy!
2. Look beyond the surface for hidden clues

Theory

According to the description, to get the flag we have to find some hidden message inside this PDF file, we should download the file first, then use the poppler-utils app that gives us the pdftotext command, so that we can get all the text on the PDF file, because when I opened the PDF it had a couple of redacted details, so we'll check that. So let's first download the file.

Solution

Let's download the PDF file and see what text it has in it:
